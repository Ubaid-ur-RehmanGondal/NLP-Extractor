\section*{Project Setup}

\begin{enumerate}[label=\textbf{US-\arabic*:}]

\item As a project manager, I want to create a new project with custom settings so that I can organize work according to project requirements.
\begin{itemize}
    \item Acceptance Criteria:
    \item Verify that the "Create Project" button opens a configuration modal.
    \item Ensure custom settings include project name, start date, and team selection.
    \item Confirm that the project is saved to the database upon clicking "Save".
\end{itemize}

\item As a project manager, I want to assign team members to projects so that they can start contributing to project tasks.
\begin{itemize}
    \item Acceptance Criteria:
    \item Verify that a list of available team members is displayed.
    \item Ensure multiple members can be selected at once.
    \item Confirm that assigned members receive an email notification.
\end{itemize}

\item As a project manager, I want to define project milestones so that I can track major deliverables and deadlines.
\begin{itemize}
    \item Acceptance Criteria:
    \item Verify that milestones can be added with a title and due date.
    \item Ensure milestones appear on the project timeline view.
    \item Confirm that completing a milestone updates the project progress bar.
\end{itemize}

\item As a project manager, I want to archive completed projects so that I can keep the project list clean and organized.
\begin{itemize}
    \item Acceptance Criteria:
    \item Verify that an "Archive" option is available for completed projects.
    \item Ensure archived projects are hidden from the active dashboard.
    \item Confirm that archived projects can be restored if needed.
\end{itemize}

\item As a project team member, I want to view all projects I'm assigned to so that I can access the work I need to do.
\begin{itemize}
    \item Acceptance Criteria:
    \item Verify that the "My Projects" tab displays only assigned projects.
    \item Ensure clicking a project opens the detailed task view.
    \item Confirm that the list updates in real-time when assignments change.
\end{itemize}

\end{enumerate}
